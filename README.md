 # Data Visualization Project Video Game Sales Analysis

## 📌 Project Overview

This project analyzes a video game sales dataset to uncover trends in global sales, game genres, gaming platforms, publishers, and critic ratings. The analysis includes data cleaning, exploratory data analysis (EDA), feature engineering, interactive visualizations, and a Streamlit dashboard.

---

## 🎯 Objectives

- Clean and preprocess the dataset.
- Perform exploratory data analysis (EDA).
- Analyze sales trends across platforms, genres, publishers, and years.
- Identify relationships between critic scores, user scores, and global sales.
- Build an interactive dashboard using Streamlit.

## 🚀 How to Run the Project

### 1. Clone the repository

```bash

```

### 2. Navigate to the project folder

```bash
cd Video_Game_Sales_Analysis
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit app

```bash
streamlit run app.py
```
